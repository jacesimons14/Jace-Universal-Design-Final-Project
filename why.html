Why Universal Design?

The web is constantly evolving, and it needs to be accessible to everyone.

  • People with disabilities need to be able to use the web just like everyone else
  • Mobile users need mobile versions of websites to be able to use them on the go
  • Old devices and slow internet need to be able to access the web too

Accessible design isn't just about disabilities, its about mobile and internet connection too, just to name a few. Universal Design is the next logical step in making information available for everyone and achieving the full potential of the internet.
